Preparation: Install node.js
1. check yarn version using cmd/powershell: yarn --version
2. install yarn using cmd/powershell: npm install --global yarn (if no version is given)
3. install using powershell: yarn add http-server
this will be added to your current folder:
		package.json
		yarn.lock

4. install using cmd: yarn add http-server
	this will be added to your current folder:
		yarn.lock

        update the package.json:

        {
            "devDependencies": {
                "http-server": "^14.1.1"
            },
            "scripts": {
                "dev": "http-server src -p 8080" (remove src if your index.js is in the main folder)
            }
        }
5. Open in live server to check using powershell: npm run dev
    NOTE: 
    http://192.168.80.101:8080 (only works here)
    http://127.0.0.1:8080 (having error here, don't use this)

------everything is working until here-----------

6. CLI installation: https://docs.netlify.com/cli/get-started/
	a. Stop server by CTRL+C
	b. run this using cmd: yarn add -D netlify-cli
	Notes: npm i -g netlify-cli , netlify , c
	Notes: go to package.json then run: ./node_modules/.bin/netlify -h (option 1)
	d. add "netlify": "./node_modules/.bin/netlify" to package.json (option 2) choose this
                    {
                "dependencies": {
                    "dotenv": "^16.3.1",
                    "http-server": "^14.1.1"
                },
                "scripts": {
                    "dev": "http-server -p 8080",
                    "netlify": "./node_modules/.bin/netlify" //Added
                },
                "devDependencies": {
                    "netlify-cli": "^15.11.0"
                }
                }
	e. run: yarn netlify -h (use this in cmd)
	OR:  npm run netlify
7. create a folder the folder: netlify.toml

		[build]

    Netlify CLI is Install: npm install -g netlify-cli
    Login to Netlify: netlify login
    Link the Site: netlify link
